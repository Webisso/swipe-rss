Swipe RSS

A multi-language RSS feed library in OPML format. Subscribe to the full feed list with a single OPML URL in any RSS reader.

Feed URL (add to your RSS/OPML client) https://webisso.github.io/swipe-rss/feeds.opml

Download OPML

Why Swipe RSS?

Curated feeds by category

Science, technology, news, entertainment, culture, sports, economy, and more — all in tidy categories.

Multiple languages

Feeds in Turkish, English, and other languages. One list, many languages.

One URL

A single OPML address for the full feed list; no need to add feeds one by one.

Repository structure

  • feeds.opml — Main OPML file (served at the feed URL above).
  • scripts/ — Utilities (e.g. feed validation).
  • .github/workflows/ — Automation (e.g. feed checks).
